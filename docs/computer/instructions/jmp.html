JMP ​

Esta instrucción salta incondicionalmente. Los flags no se modifican.

Copiará la dirección de salto en IP.

Uso ​

vonsim
JMP etiqueta

etiqueta debe ser una etiqueta que apunta a una instrucción.

Ejemplo ​

vonsim
        org 2000h
salto:  push ax
        ; --- etc ---

        jmp salto ; Válido
        jmp 2000h ; Inválido, debe ser una etiqueta
        hlt
        end

Codificación ​

00110000, dir-low, dir-high
